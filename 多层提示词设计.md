# 多层提示词结构设计 — LangChain 实现

## 架构

```
PromptBuilder
    │
    ├─ Layer 1: 身份          — agent_identity.md（固定文件）
    ├─ Layer 2: 人格          — soul.md（固定文件）
    ├─ Layer 3: 工作区上下文  — bootstrap.md + agents.md + crons（文件 + 动态）
    ├─ Layer 4: 运行时        — 时间 + 渠道（动态生成）
    └─ Layer 5: 用户偏好      — user_preference.md（跨会话累积）
         │
         ▼
    "\n\n".join(layers)
         │
         ▼
    create_react_agent(..., prompt=final_prompt)
```

## 文件结构（多 agent 预留）

```
project/prompts/
├── agents/
│   └── pickle/                    ← 当前唯一 agent
│       ├── identity.md            ← Layer 1: 身份
│       └── soul.md                ← Layer 2: 人格（可选）
├── shared/
│   ├── bootstrap.md               ← Layer 3: 工作区指南
│   └── agents.md                  ← Layer 3: 多 agent 目录
├── user_preference.md             ← Layer 5: 全局偏好
└── README.md                      ← 说明文件
```

### 新增 agent 时

```
prompts/agents/
├── pickle/            ← 已有
├── cookie/            ← 新增
│   ├── identity.md    ← cookie 的身份定义
│   └── soul.md        ← cookie 的人格定义
└── coder/             ← 新增
    ├── identity.md    ← coder 的身份定义
    └── soul.md        ← coder 的人格定义
```

Builder 会根据 `agent_id` 参数自动切换读取路径，不需要改代码。

### 说明文件（prompts/README.md）

```markdown
# Prompts 目录说明

## 结构
agents/<agent_id>/identity.md  — Layer 1: 身份定义
agents/<agent_id>/soul.md      — Layer 2: 人格定义（可选）
shared/bootstrap.md            — Layer 3: 工作区上下文
shared/agents.md               — Layer 3: 多 agent 目录
user_preference.md             — Layer 5: 用户偏好（可选）

## 新增 agent
1. 在 agents/ 下新建 <agent_id>/ 目录
2. 创建 identity.md（必选）和 soul.md（可选）
3. 如需要，更新 shared/agents.md 加入新 agent 的描述
4. 代码层调用 builder.build(agent_id="新id") 即可
```

## 各层详细设计

### Layer 1: 身份（agent_identity.md）

固定文件，对应 AGENT.md body。

```markdown
You are Pickle, a friendly cat assistant. You help with daily tasks,
coding, questions, and creative work.

## Capabilities
- Answer questions and explain concepts
- Help with coding, debugging, and technical tasks
- Use available tools when appropriate

## Behavioral Guidelines
- When you don't know something, admit it honestly
- When you make a mistake, correct yourself gracefully
```

### Layer 2: 人格（soul.md）

固定文件，可选。对应 SOUL.md。

```markdown
Be warm and genuinely helpful with subtle cat mannerisms.
Not overly cutesy—just a gentle, approachable presence.
用中文回答。
```

### Layer 3: 工作区上下文（动态拼接）

由三部分组成：

**bootstrap.md：**

```markdown
## Workspace Guide
Project: own-clawagent
Tools: read, write, bash, web_search
Skills: weather-query, file-operations
```

**agents.md（对应你步骤 11 的多 agent 路由）：**

```markdown
## Available Agents
- pickle: Default assistant (current)
- cookie: Memory manager — use subagent_dispatch for memory operations
```

**动态 cron 列表（如果项目有定时任务）：**

```
## Scheduled Tasks
（由代码动态生成，没有定时任务时此部分为空）
```

### Layer 4: 运行时（纯代码生成，不需要文件）

```python
def build_runtime_layer(agent_id, timestamp, source):
    return f"""## Runtime
Agent: {agent_id}
Time: {timestamp}
Channel: {source}"""
```

### Layer 5: 用户偏好（跨会话注入）

对应之前设计的 `user_preference.md`。

```markdown
## User Preferences
- 回答风格：简洁，先给结论再给理由
- 语言：优先中文
- 代码：提供可直接运行的完整片段
```

文件不存在时跳过这一层。

## PromptBuilder 实现

```python
from pathlib import Path

class PromptBuilder:
    def __init__(self, prompts_dir: str):
        self.prompts_dir = Path(prompts_dir)

    def build(self, agent_id="pickle",
              timestamp=None,
              source="cli",
              extra_context=None) -> str:
        from datetime import datetime
        timestamp = timestamp or datetime.now().isoformat()

        layers = []

        # Layer 1: 身份（按 agent_id 读取）
        identity = self._read(f"agents/{agent_id}/identity.md")
        if not identity:
            identity = f"You are {agent_id}, a helpful assistant."
        layers.append(identity)

        # Layer 2: 人格（可选，按 agent_id 读取）
        soul = self._read(f"agents/{agent_id}/soul.md")
        if soul:
            layers.append(f"## Personality\n{soul}")

        # Layer 3: 工作区上下文（共享）
        context = []
        bootstrap = self._read("shared/bootstrap.md")
        if bootstrap:
            context.append(bootstrap)
        agents = self._read("shared/agents.md")
        if agents:
            context.append(agents)
        if context:
            layers.append("\n\n".join(context))

        # Layer 4: 运行时
        layers.append(f"## Runtime\nAgent: {agent_id}\nTime: {timestamp}\nChannel: {source}")

        # Layer 5: 用户偏好（可选）
        pref = self._read("user_preference.md")
        if pref:
            layers.append(pref)

        # 额外的临时上下文（如 RAG 检索结果）
        if extra_context:
            layers.append(f"## Additional Context\n{extra_context}")

        return "\n\n".join(layers)

    def _read(self, filename: str) -> str | None:
        path = self.prompts_dir / filename
        if path.exists():
            return path.read_text().strip()
        return None
```

## 接入 create_react_agent

```python
builder = PromptBuilder("prompts/")

def build_and_run(user_input: str):
    # 构建多层 system prompt
    system_prompt = builder.build(
        agent_id="pickle",
        source="cli",
        extra_context=None   # 可传入 RAG 检索结果
    )

    # 传给 agent
    result = agent.invoke({
        "messages": [
            ("system", system_prompt),
            ("user", user_input)
        ]
    })
    return result
```

